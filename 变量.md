# 变量

## 区别

```sh
${xxx}: 设置在 dialplan、application、directory 中的信道变量。
```

```sh
$${xxx}: 当freeswitch.xml被编译后值将被绑定。这个绑定操作发生在启动阶段，或者是调用了 reloadxml 命令。
```

```sh
$${xxx} 预处理后将绑定到对应的值上，不可改变。
${xxx} 会在每次被执行的时候更新它的值
```

## 查看

cli 内执行

```sh
eval ${xxx}
```

或者

```sh
bin/fs_cli -H 10.0.2.15 -p 123456 -rRS -x 'eval ${xxx}'
```
